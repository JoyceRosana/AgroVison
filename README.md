# AgroVision

##  Tema
Projeto desenvolvido para o **Concurso Agrinho 2026**, inspirado no tema:  
**"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**

## Objetivo
O **AgroVision Sustentável** nasceu da ideia de mostrar como a tecnologia pode ajudar a pensar em soluções para o campo e para o meio ambiente.  
A proposta é usar programação para criar uma experiência interativa que conscientize sobre práticas agrícolas sustentáveis e o impacto positivo que elas trazem para o futuro.

O site funciona como uma **plataforma educativa**, oferecendo:

- **Simulador:** permite testar cenários de plantio e manejo do solo, mostrando os impactos das escolhas e auxiliando no aprendizado prático.  
- **Base de conhecimento:** reúne conteúdos sobre solos, culturas, sustentabilidade e tecnologias agrícolas, com linguagem simples e confiável.  
- **Interface intuitiva:** facilita a navegação e torna o aprendizado acessível.  
- **Público-alvo:** estudantes, professores, produtores rurais e todos que desejam aprender mais sobre agricultura sustentável e o cuidado com o solo.
## Tecnologias
- **[HTML](ca://s?q=Explicar_HTML_no_projeto)** → estrutura do site  
- **[CSS](ca://s?q=Explicar_CSS_no_projeto)** → estilo e identidade visual  
- **[JavaScript](ca://s?q=Explicar_JavaScript_no_projeto)** → interatividade e dinamismo

## 📷 Créditos, Referências e Uso de IA
Este projeto foi construído utilizando recursos visuais externos e suporte tecnológico, distribuídos da seguinte forma:

- Imagens de referência retiradas do **Pinterest** e **Google**, utilizadas estritamente para fins educacionais e conceituais.  
- Elementos visuais e sonoros gerados por **Inteligência Artificial**, com prompts próprios e personalizados para o tema.  
- A IA também foi utilizada como **assistente de programação**, ajudando a refinar o design, otimizar a lógica e deixar o código mais limpo.  
- Toda a arquitetura do site (HTML, CSS e JavaScript) foi integrada, testada e adaptada manualmente por mim.  

##  Regulamento
Este projeto segue as regras do **Concurso Agrinho 2026**:  
- Repositório público no GitHub  
- Uso exclusivo de **HTML, CSS e JavaScript** (sem frameworks)  
- Publicação funcional no **GitHub Pages ou Vercel**  
- Estrutura organizada e documentação clara

 O **AgroVision** é mais que um projeto de programação:  
é uma iniciativa para unir tecnologia e sustentabilidade, mostrando que o futuro do campo depende de escolhas conscientes hoje.
