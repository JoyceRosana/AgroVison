# AgroVision

##  Tema
Projeto desenvolvido para o **Concurso Agrinho 2026**, inspirado no tema:  
**"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**

## Objetivo
O **AgroVision** nasceu da ideia de mostrar como a tecnologia pode ajudar a pensar em soluções para o campo e para o meio ambiente. A proposta é usar programação para criar uma experiência interativa que conscientize sobre práticas agrícolas sustentáveis e o impacto positivo que elas trazem para o futuro.

## Tecnologias
- **[HTML](ca://s?q=Explicar_HTML_no_projeto)** → estrutura do site  
- **[CSS](ca://s?q=Explicar_CSS_no_projeto)** → estilo e identidade visual  
- **[JavaScript](ca://s?q=Explicar_JavaScript_no_projeto)** → interatividade e dinamismo
