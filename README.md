# AgroVision

## Tema

Projeto desenvolvido para o **Concurso Agrinho 2026**, inspirado no tema:

**"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**

## Objetivo

O **AgroVision Sustentável** nasceu da ideia de mostrar como a tecnologia pode ajudar a pensar em soluções para o campo e para o meio ambiente.

A proposta é usar programação para criar uma experiência interativa que conscientize sobre práticas agrícolas sustentáveis e o impacto positivo que elas trazem para o futuro.

O site funciona como uma **plataforma educativa**, oferecendo:

* **Simulador:** permite testar cenários de plantio e manejo do solo, mostrando os impactos das escolhas e auxiliando no aprendizado prático.
* **Base de conhecimento:** reúne conteúdos sobre solos, culturas, sustentabilidade e tecnologias agrícolas, com linguagem simples e confiável.
* **Interface intuitiva:** facilita a navegação e torna o aprendizado acessível.
* **Público-alvo:** estudantes, professores, produtores rurais e todos que desejam aprender mais sobre agricultura sustentável e o cuidado com o solo.

## Tecnologias

* **HTML** → estrutura do site
* **CSS** → estilo e identidade visual
* **JavaScript** → interatividade e dinamismo
